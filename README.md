# Lança Fogos

Vou criar uma aplicação interativa e completa de um **Gerador de Fogos de Artifício**.

Esta aplicação será construída num único ficheiro HTML, utilizando a Canvas API para animações fluidas e de alto desempenho, a Web Audio API para sintetizar efeitos sonoros realistas (lançamentos, explosões e estalidos) sem depender de ficheiros de som externos, e Tailwind CSS para criar uma interface de controlo moderna, elegante e totalmente responsiva.

A interface inclui controlos para personalizar:

* **Modo Automático:** Ative um espetáculo contínuo de fogos de artifício.
* **Tipos de Explosão:** Esfera clássica, Anel, Coração, Salgueiro Chorão (fios longos) e Estalidos (Dupla explosão).
* **Palete de Cores:** Arco-íris, Neon, Tons Pastel, Dourado e Cores Patrióticas.
* **Parâmetros de Física:** Gravidade, resistência do ar (vento) e tamanho da explosão.
* **Efeitos de Áudio:** Ative/desative o som com controlo de volume integrado.

Clique em qualquer parte do ecrã para lançar os seus próprios fogos de artifício personalizados!

O código do simulador de fogos de artifício foi gerado com sucesso.

### Resumo das Funcionalidades Implementadas:

1. **Física e Estilos Personalizados**: A simulação utiliza modelos de física reais (velocidade, gravidade ajustável, atrito/desaceleração e vento que empurra o fumo e as faíscas).
2. **5 Tipos de Explosão**:
* *Clássica*: Esfera uniforme tradicional.
* *Anel*: Anel perfeitamente desenhado em expansão.
* *Coração*: Desenha uma forma paramétrica de coração através de partículas.
* *Salgueiro Chorão*: Fios longos e elegantes de queda super lenta.
* *Estalidos (Crackle)*: Centelhas que sofrem uma segunda micro-explosão com pequenos sons.


3. **Áudio de Alta Qualidade Sintetizado (Web Audio API)**: Cria efeitos acústicos de subida (frequência crescente) e explosões volumétricas e de estalidos em tempo real. Não requer downloads ou recursos externos adicionais.
4. **Interface Glassmorphism de Alta Tecnologia**: Totalmente estilizada com Tailwind CSS, recolhível e fácil de usar no telemóvel ou computador.

Espero que se divirta a ver e a configurar o seu próprio espetáculo de luzes!
